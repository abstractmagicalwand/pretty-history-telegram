# Pretty History Telegram

Utility for a message history in Telegram. That converts a history log to markdown.

**Example:**

![Example](./example.png)

## Install

### Production:
1. `$ yarn prod`
2. `$ yarn start`
3. It opens your browser: `http://localhost:3000`

### Development:
1. `$ yarn dev`
2. `$ yarn start:dev`
3. It opens your browser: `http://localhost:3000`
